Step 1: Get ChatGPT Plus ($20/mo)

You need ChatGPT Plus to create Custom GPTs. The free tier and the cheaper "Go" tier won't work.

  • New to ChatGPT: Go to chatgpt.com → sign up with your personal email → subscribe to Plus
  • Already have a free account: Settings → Subscription → Upgrade to Plus
⚠️ You need Plus specifically — not the Go tier. Only Plus unlocks Custom GPT creation.

📡 ESPN API Reference

These free, no-authentication ESPN endpoints return JSON data. Your Custom GPT can fetch them via Code Interpreter — just ask it to pull live data.

Live Scores

https://site.api.espn.com/apis/site/v2/sports/basketball/mens-college-basketball/scoreboard

Scores by Date (replace YYYYMMDD)

https://site.api.espn.com/apis/site/v2/sports/basketball/mens-college-basketball/scoreboard?dates=YYYYMMDD

Rankings (AP / Coaches Poll)

https://site.api.espn.com/apis/site/v2/sports/basketball/mens-college-basketball/rankings

All Teams

https://site.api.espn.com/apis/site/v2/sports/basketball/mens-college-basketball/teams

Conference Standings

https://site.api.espn.com/apis/site/v2/sports/basketball/mens-college-basketball/standings

Game Summary (replace {eventId})

https://site.web.api.espn.com/apis/site/v2/sports/basketball/mens-college-basketball/summary?event={eventId}

Code Interpreter Example

Use Code Interpreter to fetch the latest scores from https://site.api.espn.com/apis/site/v2/sports/basketball/mens-college-basketball/scoreboard and parse the JSON to show me today's games, scores, and status.
Also useful for web search:
  • espn.com/mens-college-basketball/scoreboard — rendered scores
  • espn.com/mens-college-basketball/rankings — AP/Coaches poll
  • espn.com/mens-college-basketball/bracketology — Lunardi's projections
  • ncaa.com/march-madness-live/bracket — Official NCAA bracket
  • barttorvik.com — Free efficiency ratings (may need manual access)
  • warrennolan.com/basketball — NET rankings, Quad records

📊 How the Data Pipeline Works

Three Data Channels:

Channel 1 — ESPN API via Code Interpreter (automated): Ask "fetch today's scores from the ESPN API" and the AI writes Python code to pull live JSON data. Works for scores, rankings, team stats, standings.

Channel 2 — Web Search (semi-automated): Ask "search for the latest tournament news" and the AI searches ESPN, CBS Sports, etc. Works for injuries, bracketology, expert analysis.

Channel 3 — Manual Paste (pool-specific): Copy-paste your ESPN Tournament Challenge group standings into the chat. The AI can't log into ESPN — this is the only way to get pool-specific data.

Combined Power:

In one conversation: "Fetch today's scores" + "Search for injury news" + [paste standings] → "Write the recap email combining all of this." The AI synthesizes all three sources into one email in your voice.

⚡ Zapier Email Automation

Automate distributing your AI-generated recap emails to the whole family.

Option A: Email Trigger (Recommended)

  1. Create a free account at zapier.com
  2. Create a new Zap:
    • Trigger: "Email by Zapier" — this gives you a unique Zapier email address
    • Action: "Gmail — Send Email"
    • To/BCC: Paste the comma-separated email list above
    • Subject: Use the trigger email's subject line
    • Body: Use the trigger email's body
  3. Now your workflow: Generate recap in Pool Commissioner AI → copy → send to your Zapier trigger email → Zapier distributes to everyone

Option B: Gmail Label Trigger

  1. Same Zapier account
  2. Create a Zap:
    • Trigger: "Gmail — New Email Matching Search" with query label:pool-updates
    • Action: "Gmail — Send Email" to the full participant list
  3. Workflow: Compose in Gmail → add "pool-updates" label → Zapier auto-distributes

Option C: Scheduled Reminders

  1. Create a Zap with trigger "Schedule by Zapier"
  2. Set to fire after each tournament round (Friday evening, Sunday evening, etc.)
  3. Action: Send YOU a reminder email — "Time to generate the pool recap!"
  4. This ensures you never forget to send updates during the tournament
Pro tip: Add a second action to any Zap to log each sent email to a Google Sheet — creates an automatic record of all pool communications for next year's reference.
