💡 What This Is

This is a personal AI assistant built specifically for you and the Nitzberg Family pool. It lives inside ChatGPT as a "Custom GPT" — think of it as a chatbot that already knows our pool, all 26 participants, your writing style, and has a live connection to ESPN's data.

What It Can Do

  • 🏀 Pull live ESPN data on demand — scores, rankings, team records, rosters, box scores, conference standings. Just ask in plain English: "What are today's scores?" and it fetches them in real time.
  • 📧 Write pool emails in your voice — it studied your actual past pool emails and writes exactly like you do, including the "Hi everyone-" opener and "-Jacob" closer.
  • 📊 Run scenario analysis — paste the pool standings, and it calculates who wins under every possible tournament outcome.
  • 🏗️ Help build your bracket — analytics-driven picks using efficiency metrics, seed history, and matchup data.
  • 📋 Track all 26 participants by name — it knows every family member and never skips anyone in recaps.

How It Gets Data (Important)

Automatic (just ask): Game scores, AP rankings, team records, rosters, box scores with player stats, conference standings. Your GPT connects to a live ESPN data feed — no copy-pasting needed for any of this.

You paste ONE thing: Pool standings from ESPN Tournament Challenge (who's winning our family pool). ESPN requires a login to see pool-specific data, so the AI can't fetch that automatically. You copy-paste it from the ESPN group page — takes 10 seconds.

Everything else is automatic. You talk to it like a person, it pulls data and writes content. That's it.

Step 4: Connect Live ESPN Data (Actions) New in v2.1.0

What Are Actions?

Actions let your GPT fetch live data automatically. Instead of copy-pasting ESPN scores, your GPT calls our proxy API and gets clean, structured data in real time. Ask "What are today's scores?" and it just… gets them.

How to Add Actions

  1. In your Custom GPT editor → scroll down to the Actions section (below Capabilities)
  2. Click Create new action
  3. Set Authentication to None
  4. In the Schema box, select all existing text and delete it
  5. Paste the OpenAPI schema below (use the Copy button)
  6. You should see 8 operations appear in the list below the schema
  7. For the Privacy policy URL, enter: https://mtblackman16.github.io/pool-commissioner-ai/
  8. Click the gear icon next to each operation and make sure "Always allow" is available (it will appear when you first use the GPT)

Why Do I Still Paste Pool Standings?

ESPN Tournament Challenge requires a login to see who's winning your specific pool. The AI can fetch public ESPN data (scores, rankings, rosters) automatically, but it can't log into your ESPN account. So for pool-specific analysis — "who's in first place," "who can still win," "write the recap email" — you paste the standings table. Everything else is hands-free.

If you want to automate distribution:

Option A: Email Trigger (Recommended)

  1. Create a free account at zapier.com (free tier = 100 tasks/month — plenty for tournament emails since BCC to 22 people = 1 task)
  2. Create a new Zap:
    • Trigger: "Email by Zapier" — this gives you a unique Zapier email address
    • Action: "Gmail — Send Email"
    • To/BCC: Paste the comma-separated email list above
    • Subject: Use the trigger email's subject line
    • Body: Use the trigger email's body
  3. Now your workflow: Generate recap in Pool Commissioner AI → copy → send to your Zapier trigger email → Zapier distributes to everyone

Option B: Gmail Label Trigger

  1. Same Zapier account
  2. Create a Zap:
    • Trigger: "Gmail — New Email Matching Search" with query label:pool-updates
    • Action: "Gmail — Send Email" to the full participant list
  3. Workflow: Compose in Gmail → add "pool-updates" label → Zapier auto-distributes

Option C: Scheduled Reminders

  1. Create a Zap with trigger "Schedule by Zapier"
  2. Set to fire after each tournament round (Friday evening, Sunday evening, etc.)
  3. Action: Send YOU a reminder email — "Time to generate the pool recap!"
  4. This ensures you never forget to send updates during the tournament
Pro tip: Add a second action to any Zap to log each sent email to a Google Sheet — creates an automatic record of all pool communications for next year's reference.

The proxy at espn-ncaa-proxy.mark-016.workers.dev works for now. But if you want full independence — your own URL, your own control — here's how to deploy it yourself on Cloudflare Workers (free).

Step-by-Step

  1. Create a free Cloudflare account at cloudflare.com
  2. Left sidebar → Build → Compute → Workers & Pages
  3. Click Create → "Start with Hello World!"
  4. Name it (e.g., espn-pool-proxy)
  5. Click Deploy (creates a placeholder)
  6. Click "Edit Code" — select all the default code and delete it
  7. Paste the worker code from the GitHub repo (link below)
  8. Click Deploy
  9. ⚠️ The preview panel may spin or hang — this is normal, ignore it
  10. Your URL appears at the top (like espn-pool-proxy.yourname.workers.dev)
  11. Test: open YOUR-URL/health in your browser — you should see a JSON response
  12. Update your Custom GPT: go to Actions → edit the schema → change the servers URL to your new URL
